# 02-object-rotation

Learning exercise for **Three.js + TypeScript**, focused on object rotation and
transformations.

## Goal

Recreate a **solar system**: sun, planets orbiting around it, and their own
axial rotation, using Three.js scene graph, `Object3D` hierarchies, and
rotation/transform APIs (Euler, Quaternion, Matrix4).

## Stack

- TypeScript
- Three.js
- Parcel (bundler/dev server)

## Scripts

- `yarn start` — run the dev server
- `yarn build` — production build

## Deployment

The project will be published on **GitHub Pages**.
